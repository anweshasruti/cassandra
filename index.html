A note on the way expressions should be input

Expressions should be entered in standard mathematical infix notation. Most expressions ought to be quite simple to input using characters available on a standard keyboard. The way input is handled is similar to the LineIO mode on Casio calculators. Some examples of valid input syntax: 

sin(x)
x^2 + 3*x
ln(x)
exp(x)
sqrt(x)
(x^2 + 1)/(x - 1)
